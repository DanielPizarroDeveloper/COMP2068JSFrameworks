# 🚀 Web Project with Express and HBS - DanielPizarroDev  
  
This is a web project developed with **Node.js**, using **Express.js** as the backend framework and **Handlebars (HBS)** as the templating engine.

## 🛠️ Technologies Used  
- **Node.js**
- **Express.js**
- **Handlebars (HBS)**  
- **CSS**
